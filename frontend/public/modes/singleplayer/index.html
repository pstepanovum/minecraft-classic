Welcome to My Minecraft-like Game!

Hi, I'm Pavel. This project is a Minecraft-inspired game built with Three.js and modern web technologies.

What's New in v2.1:

  • Enhanced Block Interaction: Improved raycasting for precise block removal
  • Real-time Player Orientation: Live updates of player movements
  • Enhanced Multiplayer: Modified blocks now sync instantly with other players
  • Performance Boost: Optimized terrain generation
  • New Features: Added trees and water to enhance world generation
Key Features:
  • Procedurally generated terrain
  • Multiplayer support
  • Customizable world settings
Game Modes:
  • Online: Connect with others, sync world data
  • Offline: Play solo with local world generation

The game automatically switches between modes based on server availability.

Controls:
Movement
  • WASD to move
  • Space to jump
  • Shift to crouch
  • F to fly
Interaction
  • Mouse to look around
  • Left Click to destroy blocks
  • Right Click to place blocks
  • N to show map
